**STEP 1:: You first have to install speech_recognition
pip install SpeechRecognition
**STEP 2:: you also have to install pyaudio. Depending on which version you could find the installation wheel here: https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
pip install PyAudio-0.2.11-cp37-cp37m-win32.whl
**STEP 3:: you have to install mutagen
pip install mutagen
** when an error occurs, first get it from git clone
git clone https://github.com/quodlibet/mutagen.git
**Then ofcourse step 3 again
**STEP 4:: install text to speach
pip install gTTS


**Side note: every file is included in the folder and can be installed from there.
