[For this project you should use Python3]

********************
******Windows*******
********************
**STEP 1:: you have to install pyaudio. Depending on which version you could find the installation wheel here: https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
pip install PyAudio-0.2.11-cp37-cp37m-win32.whl
**STEP 2:: you have to install speech_recognition
pip install SpeechRecognition
**STEP 3:: you have to install pygame
pip install pygame
**STEP 4:: you have to install mutagen
pip install mutagen
[when an error occurs, first get it from git clone
git clone https://github.com/quodlibet/mutagen.git]
**STEP 5:: install text to speach
pip install gTTS

**Side note: every file is included in the folder and can be installed from there.
